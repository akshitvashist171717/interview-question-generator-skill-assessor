#  InterviewAI — Smart Hiring Platform

An end-to-end ML-powered interview preparation and candidate assessment platform,
trained on **8,167 real resumes**. Upload a resume, paste a job description,
and get tailored interview questions generated by **Google Gemini** with
**AI-evaluated answers** — through a clean, Claude-inspired interface.

---

##  Pipeline Architecture

```
Resume Upload (PDF/DOCX)
        ↓
Resume Classifier (ML)  ← TF-IDF + Logistic Regression, trained on 8,167 real resumes
        ↓
Predicted Role  (10 IT job categories)
        ↓
Skill Extraction  ← matched against a real skill vocabulary
        ↓
Job Description Analysis
        ↓
Required Skills Extraction
        ↓
Skill Matching Engine  (weighted: 70% required / 30% preferred)
        ↓
Match Score
        ↓
Question Generation  ← Google Gemini API (+ local fallback bank)
        ↓
Answer Evaluation  ← Google Gemini (0–10 score, grade, feedback)
        ↓
Dashboard
```

### Routing & navigation

This app does **not** use Streamlit's native multipage file-based routing.
Instead, `app.py` keeps a single page in `st.session_state.page` and renders
the matching `app_views/*.py` module manually. The sidebar buttons just update
that session-state key and call `st.rerun()`. This gives full control over the
nav UI (icons, active-state styling, pipeline progress, theme picker) that the
native Streamlit sidebar doesn't support.

 If you ever add a folder named `pages/` next to `app.py`, Streamlit will
auto-detect it and inject its own sidebar nav — which will conflict with the
nav above. Keep page modules under `app_views/` (or any name other than `pages`).

##  Theme Picker

Four built-in colour themes, switchable live from the sidebar — no restart needed:

| Theme | Background | Accent | Style |
|---|---|---|---|
| **Claude Warm** (default) | `#F9F8F6` | `#C96442` terracotta | Warm off-white, serif headings |
| **Midnight Blue** | `#0F1420` | `#5B8DEF` | Dark mode, blue accent |
| **Sage Garden** | `#F6F8F4` | `#5A8A6B` | Soft green, Lora serif |
| **Violet Ink** | `#FAF8FC` | `#7C5CC4` | Lavender, Fraunces serif |

Each theme is fully self-contained in `theme.py` as a dict of CSS custom
properties plus a Google Fonts pairing. Switching themes:
1. Updates `st.session_state.theme`
2. Re-renders the CSS `:root` block via `render_theme_css()`
3. All inline HTML across pages reads `var(--accent)`, `var(--bg)`, etc.
4. Plotly charts (which can't read CSS vars) pull literal hex values via
   `theme.get_active_colors()`, called fresh on every page render

To add a 5th theme: copy a block in `THEMES` inside `theme.py`, pick a Google
Fonts pairing, and it appears in the sidebar picker automatically.



| Property | Value |
|---|---|
| Algorithm | Logistic Regression (balanced class weights) |
| Features | TF-IDF, 1-3 grams, 12,000 max features |
| Training Data | 8,167 real resumes |
| Roles | 10 IT job categories |
| Validation | 5-Fold Stratified Cross-Validation |
| CV Accuracy | ~93.6% |

The model trains automatically on first run (a few seconds) and ships
pre-trained in `ml_models/` so it works immediately on extraction.

##  Gemini Integration

**Question Generation** — tailored to matched skills, skill gaps, and the exact JD text.
**Answer Evaluation** — returns score (0–10), letter grade, feedback, strengths,
improvement areas, and detected technical keywords.

**Fallback mode** (no API key): local curated question bank + length-based heuristic scoring.
